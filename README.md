## Usage/Examples

##### Installation

1. Fork the repository from Github and clone it using the following command:

`git clone https://github.com/Rorschach3/spotifyPlaylist.git`

Before starting the server, make sure you have all the necessary files and dependencies installed. You can do this by running either of the following commands:

##### `npm i`      (`npm install`)

or

##### `yarn install`

---

2. Once the installation is complete, start the server by running either of the following commands:

##### `npm start`

or

##### `yarn start`

---

3. A browser window should automatically open, indicating that the program is running on localhost port 3000. If the browser window doesn't open automatically, you can manually open a browser and go to
   http://localhost:3000/.
